# Aplicação DevOps 🚀

## 📌 Descrição

Esta é uma aplicação desenvolvida com Node.js e Express como parte da avaliação da disciplina de DevOps.
O projeto demonstra o uso de controle de versão com Git e GitHub, utilizando branches, commits e merges.

---

## ⚙️ Tecnologias utilizadas

- Node.js
- Express
- Git e GitHub

---

## 📁 Estrutura do projeto

```
src/
 ├── server.js
 └── views/
     └── index.html
```

---

## ▶️ Como executar o projeto

1. Clone o repositório:

```
git clone https://github.com/DaviLLi/aplicacao-devops.git
```

2. Acesse a pasta do projeto:

```
cd aplicacao-devops
```

3. Instale as dependências:

```
npm install
```

4. Execute o servidor:

```
node src/server.js
```

5. Acesse no navegador:

```
http://localhost:3000
```
